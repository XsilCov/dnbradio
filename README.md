# DNBRADIO

Локальний односторінковий сайт радіо з анімаційним ASCII‑фоном, стилізованими кнопками та еквалайзером (для прямих потоків).

## Робочі інструкції
Працюємо з `C:\DNBRADIO`. Прочитай `README.md` і `NOTES.md`. Цього достатньо, щоб відновити контекст.

## Файли
- `index.html` — головна сторінка (радіо‑плеєр + фон)
- `index-dark.html` — темна версія (якщо потрібно)
- `background.html` — окремий фон

## Запуск
Відкрити `index.html` у браузері (двічі клік або перетягнути в браузер).

## GitHub Pages

## Примітки
- SoundCloud відтворюється через iframe, еквалайзер до нього не застосовується.
- EQ працює лише для прямих потоків (`.mp3/.aac/.m3u8`).
